# School Management System (C++)

## Overview
An implementation of a school management system developed to demonstrate core **Object-Oriented Programming (OOP)** principles. The system manages a hierarchy of school members, including students and various types of employees.

## Key Features
- **Inheritance & Polymorphism:** Utilizes a base class structure (`pupil`, `worker`) to manage diverse objects through a unified interface.
- **Data Management:** Handles personal details, grades, and salary calculations.
- **Advanced Sorting:** Includes a custom implementation of a sorting algorithm to organize employees by salary levels.
- **Memory Management:** Efficient use of pointer arrays for dynamic object handling.

## Tech Stack
- **Language:** C++
- **Concepts:** Virtual Functions, Encapsulation, Constructors/Destructors.
